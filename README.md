# Task Tracker Backend

A simple Spring Boot REST API for tracking tasks with CRUD operations.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger / OpenAPI

## Endpoints
- `GET /api/tasks`
- `GET /api/tasks/{id}`
- `POST /api/tasks`
- `PUT /api/tasks/{id}`
- `DELETE /api/tasks/{id}`
